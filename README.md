不太喜欢那么多空格的风格，因而写了这个蹩脚的小插件：

Before:
```c
#include <bits/stdc++.h>
using namespace std;
int main()
{
    string s = "Shutheblanksup ~ ";
    auto t = s;
    for (auto i : s)
        cout << i << endl;
    while (true)
        return 0;
}
```
  
After:
```c
#include<bits/stdc++.h>
using namespace std;
int main()
{
    string s="Shutheblanksup ~ ";
    auto t=s;
    for(auto i:s)
        cout<<i<<endl;
    while(true)
        return 0;
}
```
